# csi.lk v2

Needed a refresh from the old Gatsby approach I had so going with the most simplistic build engine I can think of and removing any JS on the frontend

Written using:

- 11ty
- Typescript
- Custom JSX Factory
- PostCSS

## Quick Start

```bash
yarn
yarn dev
```

## Linting

Using:

- stylelint
- eslint

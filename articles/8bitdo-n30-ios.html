TL;DR Download firmware, flash (power+pair), connect, map

Wanted to use the Delta emulator from AltStore on my iPhone and had an old 8BitDo N30 Pro controller lying around.

This turned into a bit of a journey to get it to work:

  • Figured out I needed to downgrade the firmware of the controller to v2.0
  • Download the legacy firmware from the 8BitDo Support page | (my mirror just in case)
  • Hold the power and pair butons for 3-4 seconds then let go to allow the controler to enter firmware update mode
  • Update the firmware using the downloaded app (should only take 30seconds)
  • Find the old manual telling you to turn it on using Power + B
  • Connect via bluetooth
  • Map the controls (Settings > Controllers > Keyboard > x)

Enjoy!

Moving forward I hope they offer iOS13 support, there seems to be a lot of communication with the 8BitDo guys on the subreddit but I wouldn't hold my breath

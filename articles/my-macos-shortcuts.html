My MacOS Shortcuts

Posted on: 2020-08-02

source

My wife asked me to put together a list of common shortcuts I use constantly throughout my day, I thought others may get some value from it.

System

Key Combination Description
⌘ Cmd + C Copy to clipboard
⌘ Cmd + P Paste
⌘ Cmd + Space Alfred (replaced spotlight)
⌘ Cmd + Tab Switch between apps
⌘ Cmd + ` Switch between app windows
⌥ Opt + Space Open terminal
⌘ Cmd + ⇧ Shift + 4 Take Screenshot
⌘ Cmd + ⌥ Opt + ArrowKey Move window half left / right / up / down
⌘ Cmd + ⌥ Opt + F Full screen window

Slack

Key Combination Description
⌘ Cmd + K Open quick switcher
⌘ Cmd + ] Go forward a conversation
⌘ Cmd + [ Go back a conversation
/giphy [keyword] to create a GIF
/remind to create reminders
ArrowKeyUp + E Select the message you just posed and edit it quickly

1Password

Key Combination Description
⌘ Cmd + \ Quick fill username and password into window
⌘ Cmd + ⌥ Opt + \ Open 1 password window to search for entry (eg. Credit
⌘ Cmd + N New password

Browser

Key Combination Description
⌘ Cmd + T New tab
⌘ Cmd + ⇧ Shift + T Reopen recently closed tab
⌘ Cmd + W Close current tab
^ Ctrl + Tab Go to next tab
^ Ctrl + ⇧ Shift + Tab Go to previous tab
⌘ Cmd + (Number) Go to that tab number eg. ⌘ Cmd + 1 will take
⌘ Cmd + L Jump to url input box

Things

Key Combination Description
^ Ctrl + Space Quick todo entry
^ Ctrl + ⇧ Shift + Space Contextual quick todo entry
⌘ Cmd + T Set to 'today'
⌘ Cmd + . Complete todo
Delete Delete todo
